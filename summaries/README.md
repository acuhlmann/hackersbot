# HackerNews Summaries

This directory contains summaries generated by the GitHub Actions workflow.

Summaries are automatically generated:
- **Daily** at 8 AM UTC
- **On PR merge** to main branch
- **On-demand** via manual workflow trigger

## How to Trigger Manually

1. Go to **Actions** tab
2. Select **"HackerNews Summary"** workflow
3. Click **"Run workflow"**
4. Choose your options and run!

## File Format

- `YYYY-MM-DD_HH-MM-SS_summary.md` - Human-readable markdown
- `YYYY-MM-DD_HH-MM-SS_summary.json` - Machine-readable JSON
